Python Library
agent-recall

Drop-in persistent memory for LLM agents. Three memory layers — short-term, long-term, and episodic — with a single unified API. Framework-agnostic and local-first.

  • Short-term session buffer with auto-compression
  • Long-term semantic memory via local vector DB
  • Episodic action log — filterable by tag & time
  • Works with any LLM or agent framework

json-storyteller

A React component that transforms raw JSON into human-readable narratives using any LLM provider. Drop it in, pass your data, pick a tone — done.

  • Supports Anthropic, OpenAI, Mistral & custom providers
  • Configurable tone: narrative, analyst, casual
  • TypeScript-first with full type safety
  • Zero lock-in — bring your own LLM adapter

context-forge

Production-grade retrieval-augmented generation pipeline for React. Index files, URLs, or plain text — query with any LLM. In-memory vector store by default, fully pluggable for Pinecone, Weaviate, pgvector, and more.

  • Six-stage pipeline: load, chunk, embed, store, retrieve, generate
  • Supports Anthropic, OpenAI, Mistral & custom LLM providers
  • Custom adapters for vector stores, embeddings & chunkers
  • Cited answers with similarity scores — zero vendor lock-in

  • Sequential & parallel orchestration modes

sql-narrator

A React component that explains SQL queries — and what their results actually mean — in plain English. Multi-dialect, four tones, and works with any LLM provider.

  • Six SQL dialects: Snowflake, Postgres, MySQL, BigQuery, more
  • Four tones: narrative, analyst, casual, teacher
  • Interprets result sets in context of the query
  • Flags caveats: perf smells, ambiguous joins, dialect quirks

pipeline-explainer

Paste Snowflake CREATE TASK DDL (or any DAG as JSON), get an interactive SVG graph plus an LLM-generated narration of how the pipeline flows — fan-out, fan-in, stream conditions, finalizers, and risks.

  • Built-in Snowflake DDL parser: AFTER, WHEN, FINALIZE
  • SVG DAG with click-to-inspect per-node notes
  • Bring any graph: Airflow, Dagster, dbt — pass a JSON pipeline
  • Flow narration + risk observations from any LLM provider

tool-pilot

Task-planning AI agent with a ReAct reasoning loop and pluggable tools. Give it a goal — watch it think, pick tools, observe results, and deliver an answer.

  • ReAct loop: Think → Act → Observe → repeat
  • Built-in tools: web search, code execution, file reading
  • Create custom tools with a simple ToolDefinition interface
  • Live visual reasoning trace — every step streamed to UI
